Learning Threads User Guide

What is a Learning Thread?

A thread is an organized collection of online resources.

Adding a Resource to a Learning Thread

To add the current URL to the current learning thread, just click the Add to Thread button. You can also grab the and drag-and-drop it onto the name of the thread that you'd like to add the resource to.

Connecting Resources

Resources can be connected together to improve the organization. To connect two resources, just grab the and drag-and-drop it onto another

Importing Learning Thread(s) from File

Awesome, someone sent you a Learning Thread! Just press the button and load the file. (Hint: It's a .json file)

Importing a Public Learning Thread

Handmade Learning Threads just for you! Just press the button, type in the name, and press the Add button.

Export Thread to File

Sweet, you're ready to share how you've learned something! Just press the button and choose a file name to save the thread to. Then, you can send it to whomever you like!

View Thread in Graph Mode

This tool is in beta, but we've included it just for you. Visualize your thread by pressing the button.
